# Instalación

Para utilizar este proyecto se recomienda contar con:

- Git instalado.
- Una cuenta de GitHub.
- Acceso a una terminal como Git Bash.
- Conocimientos básicos de comandos Git.

## Pasos básicos

1. Clonar el repositorio.
2. Revisar los archivos del proyecto.
3. Consultar el historial de commits.
4. Crear ramas para nuevas funcionalidades.
